


# BigData

BigData is C++ header file. used in arithmatic operation of large data (~upto 2^32 digits)

  

## Feature

- Addition
- Subtraction
- Divide
- Multiplication
- Square/Squareroot
- Factorial
- and many more...

  
## How to use

Variable Declearation and initialization

```bash
  big x , y;
  x = "123";
  y = "456";
```

Arithematic Operation

```bash
  big x , y , z;
  x = "24";
  y = "365";
  z = x+y;
  z = x*y;
  z = x-y;
  z = x/y;
```

Changing datatype from int to big

```bash
  int x = 20;
  big y;
  y = to_big(x);
```

Conditional Operation

```bash
  big x,y;
  x ="-334";
  y = "234325"
  if(x<y){}
  if(x<=y){}
  if(x==y){}
  if(x>=y){}
  if(x>y){}
```
